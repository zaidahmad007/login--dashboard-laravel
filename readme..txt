Steps to install in local host:
  
 Step1: Unzip at the location C:/xampp/htdocs/task/
 Step2: Create a Database Name 'text-laravel'.
 Step3: In the directory Open Git Bash.
 Step4: Write the command - " php artisan migrate"
 Step5: Now Write the command - "php artisan serve"
 Step6: Open the browser and write url- "http://localhost:8000
